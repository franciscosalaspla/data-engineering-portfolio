# ETL Simple con Python

Proyecto práctico de ingeniería de datos para construir un pipeline ETL simple usando Python y Pandas.

## Objetivo

Extraer datos desde un archivo CSV, explorar su calidad, limpiar nulos y duplicados, calcular métricas de negocio y guardar resultados en CSV y Parquet.

## Stack utilizado

- Python
- Pandas
- CSV
- Parquet
- Git / GitHub
